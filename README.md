# MRC-based-Chinese-Opinion-Target-Extraction
Implementation of Chinese Opinion Target Extraction  based on Extractive MRC framework

# requirments

- python 3.7
- paddlepaddle-gpu-2.1.0
- paddlenlp-2.0.6
